## 关于《DAMN》/ About

《DAMN》（Discovery and Advancement of Modern Nature），中文名《彻诞》，创办于2026年，是一本致力于收录**严肃荒诞研究**的开放获取学术期刊。

**我们相信：世界上最伟大的发现，往往诞生于最荒诞的提问。**

### 核心理念 / Philosophy

> - 课题可以荒诞，但**思维必须严谨**
> - 表达可以有趣，但**逻辑必须自洽**
> - 形式可以解构，但**绝不允许造假**

### 评论区文化 / Comment Culture

《彻诞》鼓励有质量的讨论：

- 🔍 **认真审视：** 像真正的学术审稿一样认真，指出方法论漏洞、逻辑矛盾、或者"这个课题还不够荒诞"
- 💬 **继续扯诞：** 建设性地提出更荒诞的研究方向，或补充更妙的角度

## 投稿指南 / Submission Guidelines

### 征稿范围 / Scope

《DAMN》欢迎来自**一切学科领域**的投稿，包括但不限于：自然科学、工程技术、社会科学、人文艺术、哲学思辨、生活科学、游戏研究、美食学、影视分析、武侠学、神话学、网络文化学……以及任何尚未被命名的学科。

我们尤其欢迎以下类型的研究：

- ⚗️ 用科学方法解析日常生活中的荒诞现象——例：为什么微信消息"在吗"的回复率与发送时间呈非U型曲线分布
- 🔬 用荒诞视角重新审视严肃学术问题——例：基于博弈论的宿舍空调温度战争研究
- 🧬 跨学科融合的奇思妙想——例：用流体力学分析食堂排队动力学
- 📊 对社会现象的解构式研究——例："开会时长"与"实际产出"的负相关性实证分析
- 💡 任何让你在深夜灵光一闪、拍案叫绝的研究设想

如果你的灵光一闪发生在凌晨，请务必先记下来再睡觉。根据我们的非正式统计，凌晨灵感的存活率不足12%。

### 投稿格式规范 / Format Requirements

投稿内容可以采用**中文**或**英文**。

**文件要求：**

- ✦ 接受格式：**Word（.docx）、Markdown、PDF** 等格式
- ✦ 图表请尽量清晰，分辨率不低于"能看清楚"的程度
- ✦ 公式推荐使用 LaTeX 排版，手写拍照也行——只要你的字比医生的处方好认

**论文结构要求：**

| 序号 | 内容 | 说明 |
|:---:|------|------|
| 01 | **标题** | 得有。好标题的标准是：让导师看到会沉默，让朋友看到会转发，让陌生人看到会点进来。 |
| 02 | **摘要** | 必须提供，用一段话概括研究的核心发现。 |
| 03 | **引言** | 说明研究背景和动机。 |
| 04 | **研究过程** | 可以是数据分析、实验、调查、模型构建、理论推导等任何正规方法。 |
| 05 | **结果与讨论** | 你发现了什么。有图表更佳。 |
| 06 | **结论** | 一句话总结你的发现，并展望未来研究方向。 |
| 07 | **参考文献** | 如果引用了他人的资料，请正规标注。引用其他《DAMN》文章也完全可以，这会让我们非常感动。 |

### 投稿方式 / How to Submit

1. **将稿件发送至** damnjournal@163.com，邮件标题注明 **"《DAMN》投稿"**
2. **清晰标明作者信息：** 第一作者、共同第一作者、第二作者、第三作者……允许有通讯作者，并提供每位作者的单位
3. **建议字数：2,000 - 10,000 字** ——太短不足以论证，太长则说明你可能真的发现了什么——那建议你投正经期刊

> ⚠️ **署名注意：** 挂名、单位的时候，一定要经得本人和单位同意！不建议挂靠正经单位，我们害怕律师函。
>
> 特别提醒：请勿将导师列为通讯作者但不告知导师。如果你非要挂，建议先确认导师的幽默感评级在A级以上。

✅ 免费投稿 · ✅ 在线发表 · ✅ 支持一稿多投 · ✅ 不收取任何费用

如果你的文章同时被《DAMN》和《Nature》录用，我们会以你为荣。

### 审稿流程 / Review Process

稿件经编辑初审后，将由编委会重点审查：

- 🎯 **新颖有趣** ——平庸是唯一的原罪
- 🔗 **严谨有逻辑** ——荒诞的是课题，不是方法
- 🚫 **不造假不抄袭** ——零容忍

**审稿周期：我们将尽快完成审稿。** 如果超过一周没有回复，大概率是我们的邮箱又被垃圾邮件过滤器误伤了，请再发一遍。每一个投稿无论录用与否，我们都一定会给予回复的！

### 内容底线 / Content Policy

以下内容将被**直接拒稿**，不予讨论：

- ✕ 一切形式的**学术造假、数据编造、抄袭剽窃**。我们开玩笑，但不开骗局。
- ✕ 涉及**人身攻击、地域歧视、性别歧视**等恶意内容。幽默的边界是温和的冒犯，而非伤害。
- ✕ 涉及**政治敏感、违法违规**的内容。我们想长久地活下去。
- ✕ **AI一键生成的水文**。你可以用AI辅助，但核心创意和思考必须是你自己的。
